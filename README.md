# linconclass-linkbio

Landing page premium/link bio para o músico Lincon Class, criada com HTML, TailwindCSS via CDN e JavaScript puro.

## Estrutura

- `index.html`
- `assets/script.js`
- `assets/favicon.svg`
- `assets/lincon-class-cover.svg`

## Como executar

Abra `index.html` no navegador ou publique a pasta em qualquer serviço de hospedagem estática.

## Personalizacao

Atualize os links em `index.html` para apontar para os perfis oficiais definitivos de Lincon Class.
